# _raw — Material Bruto (Ingest)

Inbox do cérebro. Notas, transcripts, inspirações e insights ainda não processados.

## Arquivos e Pastas

- [[notas-soltas]] — notas avulsas do lead
- `inspiracoes/` — material externo capturado (prints, links, swipe files)
- `insights/` — sacadas pontuais
- `transcripts/` — transcrições de áudios, vídeos, calls

## Fluxo INGEST

1. Material entra em `_raw/`.
2. Gaios processa: extrai fatos/decisões/claims.
3. Distribui para [[02-areas]] e/ou [[03-resources]] relevantes.
4. Atualiza páginas relacionadas com wikilinks.
5. Move o original para `_raw/{ano}-processados/` ou deleta.
6. Loga em [[changelog]].

Ver [[schema]] para regras de INGEST.
