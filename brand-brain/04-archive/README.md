# 04-archive — Projetos Concluídos

Projetos finalizados, agrupados por trimestre.

## Convenção

`YYYY-qN/{nome-do-projeto}/`

Exemplo: `2026-q2/2026-05-landing-page-curso-x/`

## Regras

- Mover a pasta inteira de `01-projects/` para cá quando o projeto for concluído.
- Antes de arquivar, promover:
  - Decisões vencedoras → [[02-areas]] correspondente.
  - Assets reutilizáveis → [[03-resources]].
- Logar em [[changelog]].
