# Promptfy OS — Guia de Instalação

> Este guia te leva do zero ao primeiro projeto em menos de 10 minutos. Sem terminal. Sem código.

---

## O Que Você Vai Precisar

| Item | Por quê |
|---|---|
| Um computador (Windows, Mac ou Linux) | Onde tudo vai rodar |
| Conta no GitHub (grátis) | Pra autenticar o Claude |
| Conta na Anthropic / Claude Code | Pra usar a IA |
| Esta cartilha aberta | Pra seguir os passos |

⏱ **Tempo total estimado:** 10 minutos

---

## PASSO 1 — Instale o VSCode

O VSCode é o editor onde o Claude Code roda.

1. Abra seu navegador
2. Acesse: **https://code.visualstudio.com**
3. Clique no botão grande de download (ele detecta seu sistema)
4. Instale normalmente (próximo, próximo, concluir)

✅ **Pronto quando:** você consegue abrir o VSCode e ver a tela de boas-vindas.

---

## PASSO 2 — Instale a Extensão Claude Code

1. Abra o VSCode
2. No menu lateral esquerdo, clique no ícone de **quadradinhos** (Extensions)
3. Na barra de busca, digite: **Claude Code**
4. Clique em **Install** na extensão oficial da Anthropic
5. Aguarde a instalação terminar

✅ **Pronto quando:** aparece um novo ícone do Claude na barra lateral.

---

## PASSO 3 — Faça Login

1. Clique no ícone do **Claude** na lateral
2. Você vai ver um botão: **Sign in**
3. Clique e escolha entrar com sua conta da Anthropic
4. Se preferir, conecte também seu GitHub para facilitar o uso

✅ **Pronto quando:** o painel do Claude mostra que você está logado.

---

## PASSO 4 — Peça pro Claude Instalar o Promptfy OS

Aqui é a parte mágica. Você não precisa rodar nenhum comando no terminal.

1. No VSCode, abra o Claude Code (atalho: **Ctrl + Esc** no Windows, **Cmd + Esc** no Mac)
2. **Copie e cole exatamente o texto abaixo:**

```
Por favor, instale o Promptfy OS na minha área de trabalho.
Clone o repositório https://github.com/rebelo2024/promptfyos
para a pasta Promptfy-OS no Desktop. Depois me avise quando terminar.
```

3. Pressione Enter
4. Aguarde 1 a 2 minutos enquanto o Claude:
   - Cria a pasta `Promptfy-OS` na sua área de trabalho
   - Baixa todos os arquivos
   - Configura o sistema

✅ **Pronto quando:** o Claude responde algo como "Promptfy OS instalado em Desktop/Promptfy-OS. Abra a pasta e digite 'Olá Gaios' para começar."

---

## PASSO 5 — Abra a Pasta e Acorde o Gaios

1. No VSCode, clique em **File → Open Folder**
2. Navegue até a sua área de trabalho
3. Selecione a pasta **Promptfy-OS**
4. Clique em **Selecionar Pasta**
5. Abra o Claude Code novamente (Ctrl/Cmd + Esc)
6. **Digite simplesmente:**

```
Olá Gaios
```

7. Pressione Enter

🌱 **A partir daqui o Gaios assume.** Ele vai te perguntar o que você quer criar e te conduzir do zero até o produto pronto.

---

## O Que Esperar Daqui em Diante

```
🌱 Gaios aqui. O que você quer criar hoje?

VOCÊ: quero criar um saas para barbearias

🌱 Beleza! Antes de chamar o time, me responde 3 coisinhas:
   • Você já tem preço e oferta definidos? (sim/não)
   • Você já tem nome de marca? (sim/não)
   • Você já tem identidade visual? (sim/não)

VOCÊ: não, não, não

🌱 Perfeito. Vou chamar nessa ordem:
   1. Hormozi (oferta e preço)
   2. Brand (nome e posicionamento)
   3. Design (identidade visual)
   4. Storytelling (narrativa)
   5. Copy (página de vendas)
   
   Depois disso, acordo o Hefaisto pra construir. Posso seguir?
```

**Você só precisa responder em linguagem natural.** O Gaios cuida de tudo.

## Problemas Comuns

### "O Claude não conseguiu clonar o repositório"
- Verifique se está logado no GitHub
- Verifique sua conexão com a internet
- Tente o comando de novo

### "Olá Gaios não funciona"
- Confirme que você abriu a pasta **Promptfy-OS** (não o Desktop inteiro)
- Confirme que tem um arquivo `CLAUDE.md` na raiz da pasta
- Feche e reabra o Claude Code

### "Estou perdido"
- Releia este guia desde o passo 1
- Ou peça suporte: {seu canal de suporte}
